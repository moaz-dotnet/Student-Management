# Student Management
## class vs struct vs record
الكلاس هو Reference Type، بيتخزن في الـ Heap، والمتغير اللي بيشاور عليه بيحمل مرجع (Reference) لمكانه في الذاكرة.

 الـ struct فهو Value Type، بيتخزن في الـ Stack، والمتغير نفسه بيحتوي القيمة مباشرة.

الكلاس بيدعم الوراثة (Inheritance)، لكن الـ struct ما بيدعمهاش.

بعد كده، جت الـ record علشان تحل مشكلة المقارنة في الكلاسات.

لأن في الكلاس، لما بنقارن كائنين (==)، المقارنة بتكون على العنوان في الذاكرة ، مش على البيانات اللي جوّه الكائن.
عشان كده بيدي false لو قرنت two object ليهم نفس القيمه
